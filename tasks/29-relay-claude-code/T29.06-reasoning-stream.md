**Status:** Not started
**EPIC:** [29-relay-claude-code](./EPIC.md)
**PRD Requirement:** FR5
**Depends on:** T29.04

## Objective
Verify that Claude Code's extended thinking events stream correctly to the mobile app without leaking raw thinking tags into message content.

## Details
- Trigger extended thinking in Claude Code (e.g., complex reasoning prompts)
- Verify `agent_thought_chunk` events stream to the Relay in real time
- Confirm no `<think>` tag fragments appear in `agent_message_chunk` content
- Confirm the mobile app receives structured thinking events and renders them appropriately

## Acceptance Criteria
- [ ] `agent_thought_chunk` events are emitted during extended thinking
- [ ] Thinking events stream in real time (not buffered until completion)
- [ ] No `<think>` tag fragments leak into `agent_message_chunk` content
- [ ] Mobile app receives and can display structured thinking events
- [ ] Reasoning and message content are cleanly separated

## Notes
- This is critical for UX — leaked thinking tags would be confusing to users
- Compare behavior with OpenClaw's reasoning stream to ensure parity
