## Goal

Define a Dart sealed class hierarchy for ACP ContentBlock types, replacing the `ArtifactEvent`/`ArtifactType` model. The sealed class enables exhaustive `switch` dispatch and MIME-type-based rendering. Each variant maps 1:1 to an ACP content block type from the spec.

## Context

**Current model** at `conversation_state.dart:125-297`: `ArtifactEvent` is a flat class with nullable fields for every artifact type (diff, code, markdown, searchResults, error). `ArtifactType` enum has 7 values. Adding a new type requires touching the enum, the factory method, and every switch that dispatches on it.

**ACP ContentBlock types** (from `docs/specs/acp-protocol/content.md`):

| Type | Key fields | MIME association |
|---|---|---|
| `text` | `text`, `annotations` | `text/plain`, `text/markdown` |
| `image` | `data` (base64), `mimeType` | `image/*` |
| `audio` | `data` (base64), `mimeType` | `audio/*` |
| `resource` (text) | `resource.uri`, `resource.text`, `resource.mimeType` | varies |
| `resource` (blob) | `resource.uri`, `resource.blob`, `resource.mimeType` | varies |
| `resource_link` | `uri`, `name`, `mimeType`, `size` | varies |

**Tool call content types** (from `docs/specs/acp-protocol/tool-calls.md`):

| Type | Key fields |
|---|---|
| `content` | wraps a standard ContentBlock |
| `diff` | `path`, `oldText`, `newText` |
| `terminal` | `terminalId` |

The sealed class must cover both sets: standard content blocks AND tool call content types.

## Implementation

### 1. Define sealed class hierarchy (`apps/mobile/lib/models/content_block.dart`)

New file:

```dart
sealed class ContentBlock {
  const ContentBlock();
  factory ContentBlock.fromJson(Map<String, dynamic> json);
}

class TextContent extends ContentBlock {
  final String text;
  final String? mimeType;  // text/plain, text/markdown, text/x-diff, etc.
  const TextContent({required this.text, this.mimeType});
}

class ImageContent extends ContentBlock {
  final String data;       // base64
  final String mimeType;   // image/png, image/jpeg, etc.
  final String? uri;
  const ImageContent({required this.data, required this.mimeType, this.uri});
}

class AudioContent extends ContentBlock {
  final String data;       // base64
  final String mimeType;   // audio/wav, audio/mp3, etc.
  const AudioContent({required this.data, required this.mimeType});
}

class ResourceContent extends ContentBlock {
  final String uri;
  final String? mimeType;
  final String? text;      // text resource
  final String? blob;      // blob resource (base64)
  const ResourceContent({required this.uri, this.mimeType, this.text, this.blob});
}

class ResourceLinkContent extends ContentBlock {
  final String uri;
  final String name;
  final String? mimeType;
  final String? title;
  final String? description;
  final int? size;
  const ResourceLinkContent({required this.uri, required this.name, ...});
}

class DiffContent extends ContentBlock {
  final String path;
  final String? oldText;
  final String newText;
  const DiffContent({required this.path, this.oldText, required this.newText});
}

class TerminalContent extends ContentBlock {
  final String terminalId;
  const TerminalContent({required this.terminalId});
}

class RawContent extends ContentBlock {
  final Map<String, dynamic> json;
  const RawContent({required this.json});
}
```
