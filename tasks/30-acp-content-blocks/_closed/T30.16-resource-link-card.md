# Task 30.16: ResourceLinkCard

**Epic:** 30 — ACP Content Blocks
**Status:** [x]
**Depends on:** T30.07, T30.08
**Blocks:** none

## Goal

Build a `ResourceLinkCard` widget that displays ACP `resource_link` content blocks as a metadata card showing name, MIME type, size, and description. Fetch/download deferred to Epic 31 (Resource Delivery) — this task is metadata display only.

## Context

ACP `resource_link` content block:
```json
{
  "type": "resource_link",
  "uri": "file:///home/user/document.pdf",
  "name": "document.pdf",
  "mimeType": "application/pdf",
  "size": 1024000
}
```

Resource links are references to fetchable resources. Without Epic 31's delivery infrastructure, the client can only display metadata. When Epic 31 lands, the card gains a "Download" or "View" action.

## Implementation

### 1. ResourceLinkCard widget (`apps/mobile/lib/widgets/renderers/resource_link_card.dart`)

Card showing:
- File icon (derived from mimeType if possible)
- `name` (primary text)
- `mimeType` (secondary text)
- `size` (formatted: "1.0 MB")
- `description` (if present)
- `uri` (truncated, shown as tertiary text)
- Placeholder action button: "Download (coming soon)" or disabled state

```dart
class ResourceLinkCard extends StatelessWidget {
  final ResourceLinkContent block;
  const ResourceLinkCard({required this.block});
}
```

### 2. Size formatting

Helper for human-readable sizes:
```dart
String formatBytes(int bytes) {
  if (bytes < 1024) return '$bytes B';
  if (bytes < 1024 * 1024) return '${(bytes / 1024).toStringAsFixed(1)} KB';
  return '${(bytes / (1024 * 1024)).toStringAsFixed(1)} MB';
}
```

### 3. Register with RendererRegistry

`ResourceLinkContent` dispatches structurally (by sealed class type). In the registry:
```dart
// Structural dispatch in RendererRegistry.build():
if (block is ResourceLinkContent) return ResourceLinkCard(block: block);
```

### 4. Tests

- Resource link renders name, mimeType, formatted size
- Missing optional fields (size, description) handled gracefully
- Size formatting: bytes, KB, MB
