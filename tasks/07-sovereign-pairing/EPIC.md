# Epic: Sovereign Pairing & Hub Integration (07-sovereign-pairing)

Enable secure, high-competence pairing between the Fletcher mobile app and a self-hosted Heirloom Hub (OpenClaw).

## Problem Statement
App Store users start with a "blank slate" Fletcher app. They need a secure way to import local Hub configurations (Tailscale IPs, API tokens, Room IDs) without manual typing or relying on a centralized cloud.

## Design Philosophy
- **Vessel Key:** A self-contained configuration payload generated by the Hub.
- **Physical Handshake:** Pairing requires physical proximity (scanning a QR code or OCR from the Hub terminal).
- **Competence-First:** The process should feel like "connecting gear" rather than a consumer wizard.
- **Sovereign:** All pairing data stays local to the user's devices.
